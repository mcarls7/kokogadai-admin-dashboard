## 2026-05-30 - [Contribution UX in Boilerplate Repos]
**Learning:** In repositories lacking an established UI, the README and contribution process (like PR templates) become the primary user interface. Micro-UX improvements should focus on documentation scannability and developer experience (DX).
**Action:** Use emoji-prefixed headers, navigation anchors, and compact PR checklists to enhance the 'Contribution UX' of boilerplate projects.
