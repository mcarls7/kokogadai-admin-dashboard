## 2026-05-23 - Enhancing Contribution UX for Boilerplate Projects
**Learning:** In repositories that are in an early boilerplate state, the README and contribution workflows are the primary "User Interface" for developers. Improving these provides immediate UX value by reducing friction for the first set of contributors.
**Action:** Implement navigation anchors, "Back to top" links in README, and a Pull Request template with an accessibility checklist to establish high UX standards from the start.
