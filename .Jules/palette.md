# Palette's Journal - Critical UX & Accessibility Learnings

## 2026-05-30 - Improving Contribution UX in Boilerplate Repositories
**Learning:** In repositories lacking a traditional UI, the documentation (README) and the contribution workflow (PR templates) serve as the primary user interface. Scannability, clear navigation, and accessibility checklists are critical for a positive developer experience (DX).
**Action:** Use emoji-prefixed headers, quick navigation bars with explicit HTML anchors, and concise UX checklists in PR templates to reduce friction.
