## 2026-05-30 - Contribution UX in Boilerplate Repositories
**Learning:** In initial boilerplate repositories where no UI exists, the README and PR template serve as the primary "user interface" for developers. Improving their scannability and structure is a valid micro-UX enhancement that improves Developer Experience (DX).
**Action:** Use emoji-prefixed headers, navigation anchors, and explicit contribution checklists to reduce friction for early contributors.
