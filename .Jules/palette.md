## 2026-05-10 - [Contribution UX in Boilerplate Projects]
**Learning:** In repositories starting from a blank slate, the primary "User Interface" is the developer experience. Providing structured contribution guides and scannable documentation is the first step towards a healthy UX-focused culture.
**Action:** Implement PR templates and README navigation early to set a standard for accessibility and clarity before the first UI component is even built.
