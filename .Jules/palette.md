# Palette's Journal - Critical UX/Accessibility Learnings

## 2026-05-30 - Contribution UX in Boilerplate Repositories
**Learning:** For repositories in their initial boilerplate state (no code, only a README), the README and contribution workflow (PR templates) are the primary "User Interface" for developers. UX improvements here should focus on scannability, clear navigation, and setting standards for future work.
**Action:** Use emoji-prefixed headers, explicit HTML anchors for "Back to top" links, and structured PR templates with UX/Accessibility checklists.
