## 2026-05-30 - Contribution UX in Boilerplate Projects
**Learning:** In repositories lacking an established UI, the README and PR templates serve as the primary user interface. Improving their scannability and structure significantly enhances the Developer Experience (DX) and contribution flow.
**Action:** Use emoji-prefixed headers, explicit HTML anchors for navigation, and clear checklists in PR templates to guide contributors toward accessible and intuitive code.
