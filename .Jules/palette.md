## 2026-05-10 - Contribution UX in Empty Repositories
**Learning:** In repositories that are just starting (boilerplate/empty), the README and repository structure are the primary user interfaces. Improving these is "Contribution UX" (DX). Scannability and navigation are key.
**Action:** Use explicit HTML anchors (e.g., `<a name="top"></a>`) for "Back to top" links to ensure cross-platform compatibility and consistent navigation.
