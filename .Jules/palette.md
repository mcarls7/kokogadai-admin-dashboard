## 2026-05-23 - Improving Contribution UX in Boilerplate Projects
**Learning:** In projects that are in an early boilerplate stage (lacking a UI), the primary "user interface" is the documentation and the contribution workflow. Scannability, clear navigation anchors, and explicit PR checklists significantly reduce friction for early contributors.
**Action:** Implement scannable README structures with emoji headers, quick navigation bars, and explicit HTML anchors for stable "Back to top" links.
