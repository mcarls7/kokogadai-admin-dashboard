# Palette's Journal

## 2026-05-30 - Contribution UX in Boilerplate Projects
**Learning:** In repositories that are in an early boilerplate state without a UI, the "User Interface" for developers is the documentation and the contribution workflow. Scannability and clear navigation in the README are critical UX components.
**Action:** Use explicit HTML anchors and navigation bars in READMEs of early-stage projects to improve the developer experience from day one.
