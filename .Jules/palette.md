## 2026-05-30 - Contribution UX in Boilerplate Repositories
**Learning:** In repositories that lack an established UI (boilerplate/initial state), the README and Pull Request templates serve as the primary "User Interface" for developers. Improving the scannability and navigation of these documents directly impacts the "Contribution UX".
**Action:** Prioritize scannability using emoji-prefixed headers, navigation anchors, and "Back to top" links. Implement PR templates with accessibility checklists to set quality standards from the start.
