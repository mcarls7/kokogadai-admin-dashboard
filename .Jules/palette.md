## 2026-05-30 - Contribution UX in Boilerplate Projects
**Learning:** In repositories lacking a UI, the README and contribution workflow (like PR templates) ARE the user interface for developers. Enhancing scannability and structure reduces friction for future contributors.
**Action:** Use explicit HTML anchors and concise navigation bars in READMEs to improve DX (Developer Experience) when the codebase is in its early stages.
