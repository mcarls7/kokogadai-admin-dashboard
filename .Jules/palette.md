## 2026-05-30 - Contribution UX in Boilerplate Projects
**Learning:** In repositories that are in an initial boilerplate state with no UI yet, the README and contribution flow are the primary interfaces. Enhancing these is a form of 'Contribution UX' or 'Developer Experience' that sets the tone for the project's quality.
**Action:** Use scannable documentation patterns (navigation bars, anchors) and PR templates to establish high UX standards from day one.
