## 2026-05-30 - Contribution UX in Boilerplate Projects
**Learning:** In repositories with no application code, the README and PR templates serve as the primary "UI" for contributors. Enhancing these with navigation anchors and clear checklists significantly improves the Developer Experience (DX).
**Action:** Use emoji-prefixed headers and explicit HTML anchors in READMEs for better scannability and consistent navigation.
