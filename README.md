## 🚀 Quick Start

Currently, this project is in its initial setup phase.

```bash
# Clone the repository
git clone https://github.com/mcarls7/kokogadai-admin-dashboard.git

# Install dependencies (ensure pnpm is installed)
pnpm install
```

<a name="development"></a>
## 💻 Development

Use the following commands to maintain code quality and verify your changes:

- `pnpm test`: Runs the Vitest suite.
- `pnpm lint`: Checks TypeScript and ESLint rules.
- `pnpm format`: Auto-formats code with Prettier.
- `pnpm build`: Verifies the production build.
