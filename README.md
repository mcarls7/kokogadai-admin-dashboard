<a name="accessibility"></a>
## ♿ Accessibility
Accessibility is a core pillar of this project. We adhere to the following standards:
- **Semantic HTML**: Using correct tags for better screen reader support.
- **Keyboard Navigation**: Ensuring all interactive elements are reachable via `Tab`.
- **Focus States**: Clear visual indicators for currently focused elements.
- **ARIA Labels**: Providing context for icon-only buttons and complex widgets.
