## ♿ Accessibility
Accessibility is a core priority for this project:
- **Semantic HTML**: Using correct elements for their intended purpose.
- **Keyboard Navigation**: Ensuring all features are accessible via keyboard.
- **ARIA Compliance**: Providing necessary context for screen readers.
