# Kokogadai Admin Dashboard 🎨

A user-centric admin dashboard designed with a focus on micro-UX delights and uncompromising accessibility.

## 🚀 Overview

The **Kokogadai Admin Dashboard** is a management interface designed to be intuitive, responsive, and pleasant to use. We believe that good UX is invisible—it just works. This dashboard aims to provide a seamless experience for administrators, ensuring that every interaction feels smooth and every piece of information is accessible.

## ♿ UX & Accessibility First

At Kokogadai, accessibility is not an optional feature; it is a core requirement. We adhere to the following principles:

- **Semantic HTML:** Using appropriate HTML elements for their intended purpose.
- **Keyboard Navigation:** Ensuring all interactive elements are reachable and usable via keyboard.
- **ARIA Labels:** Providing descriptive labels for icon-only buttons and complex components.
- **Color Contrast:** Maintaining high contrast ratios for readability.
- **Micro-UX Delights:** Adding small touches like loading states, hover effects, and helpful empty states to enhance the user journey.

## 🛠️ Contribution Guidelines

We welcome contributions that align with our UX philosophy! When submitting a PR, please ensure:

1.  **Accessibility Check:** Run an accessibility audit on your changes.
2.  **Keyboard Support:** Verify that all new features are fully keyboard-navigable.
3.  **Visual Polish:** Check for consistent spacing, alignment, and hover states.
4.  **Descriptive PRs:** Clearly explain the UX improvement and its benefit to the user.
