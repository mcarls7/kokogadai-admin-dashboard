## ♿ UX & Accessibility First

- **Semantic HTML:** Appropriate elements for intended purposes.
- **Keyboard Navigation:** Full accessibility via keyboard.
- **ARIA Labels:** Descriptive labels for icon-only components.
- **Color Contrast:** High contrast ratios for readability.
- **Micro-UX:** Loading states, hover effects, and helpful feedback.
