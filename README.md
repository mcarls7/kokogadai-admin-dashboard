## 🏠 <a id="overview"></a> Overview
A modern, accessible admin dashboard for Kokogadai. Designed with a focus on intuitive user interactions and seamless management.

## ♿ <a id="accessibility"></a> Accessibility
- **Semantic HTML**: Using correct elements for their intended purpose.
- **Keyboard Nav**: Full support for tab-based navigation and clear focus states.
- **ARIA Labels**: Descriptive labels for all icon-only interactive elements.
