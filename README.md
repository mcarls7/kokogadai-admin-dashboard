<a name="-accessibility"></a>
## ♿ Accessibility
Accessibility is a core requirement, not a feature. We strive to meet WCAG 2.1 AA standards.
- **Semantic HTML:** Correct elements for intended purposes.
- **Keyboard Navigation:** Full support for tab and enter keys.
- **ARIA Labels:** Descriptive labels for icon-only components.
- **Focus States:** High-visibility focus indicators for all interactive elements.
