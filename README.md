<a name="accessibility"></a>
## ♿ Accessibility

Accessibility is at the heart of our design philosophy:
- **Semantic HTML**: Ensures structure is clear to screen readers.
- **Keyboard Nav**: Full functionality available via keyboard.
- **Color Contrast**: AAA/AA standards for readability.
