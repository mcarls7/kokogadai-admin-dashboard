<a name="ux"></a>
## ♿ Accessibility & UX
We prioritize accessibility in every component.

- **Keyboard Navigation**: All interactive elements must be reachable and usable via keyboard.
- **ARIA Labels**: Use descriptive ARIA labels for icon-only buttons and complex components.
- **Color Contrast**: Ensure all text meets WCAG AA contrast standards.
- **Visual Feedback**: Provide clear focus states and loading indicators.
