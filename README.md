<a name="accessibility"></a>
## ♿ Accessibility
Accessibility is a core requirement of this project.
- **Semantic HTML:** We use the right tags for the right jobs.
- **Keyboard Navigation:** Every interactive element must be reachable via keyboard.
- **Visual Feedback:** Clear focus indicators and hover states are mandatory.
