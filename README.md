## 🚀 Overview
The **Kokogadai Admin Dashboard** is a management interface designed to be intuitive, responsive, and pleasant to use. We believe that good UX is invisible—it just works. This dashboard aims to provide a seamless experience for administrators, ensuring that every interaction feels smooth and every piece of information is accessible.

## ✨ UX Philosophy
- **Users notice the little things:** Micro-interactions matter.
- **Smooth interactions:** Transitions and feedback should be intentional and fluid.
- **Clarity over cleverness:** The interface should guide the user effortlessly.

## ♿ Accessibility First
Accessibility is a core requirement, not a feature. We follow these standards:
- **Semantic HTML:** Using appropriate elements for their intended purpose.
- **Keyboard Navigation:** Full support for tab order and focus states.
- **ARIA Labels:** Mandatory for icon-only buttons and interactive elements.
- **Color Contrast:** High readability for all users.
