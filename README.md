<a name="overview"></a>
## 🔭 Overview
This project provides a comprehensive administrative interface designed with a focus on user experience and accessibility.

<a name="accessibility"></a>
## ♿ Accessibility
We believe accessibility is not optional. This dashboard is built to be:
- **Keyboard Navigable:** Full support for tab order and focus states.
- **Screen Reader Friendly:** Semantic HTML and proper ARIA labeling.
- **High Contrast:** Ensuring readability for all users.

<a name="setup"></a>
## 🛠️ Setup
To get started with development:

1. **Install Dependencies:**
   ```bash
   pnpm install
   ```

2. **Start Development Server:**
   ```bash
   pnpm dev
   ```

3. **Run Tests:**
   ```bash
   pnpm test
   ```
