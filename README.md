<a name="accessibility"></a>
## ♿ UX & Accessibility First

Accessibility is not optional. This project follows strict UX guidelines:
- **Semantic HTML:** Using the right elements for the right job.
- **Keyboard Navigation:** Ensuring all features are accessible via keyboard.
- **ARIA Labels:** Providing descriptive labels for icon-only components.
- **Color Contrast:** Maintaining high contrast ratios for readability.
- **Micro-UX:** Adding loading states, hover effects, and helpful feedback.
