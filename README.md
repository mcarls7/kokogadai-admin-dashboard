## ♿ Accessibility
Accessibility is a core requirement, not a feature. We commit to:
- **Semantic HTML:** Using the correct elements for their intended purposes.
- **Keyboard Navigation:** Ensuring full support for keyboard users.
- **ARIA Labels:** Providing descriptive labels for all interactive elements.
